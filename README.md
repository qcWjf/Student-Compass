# Student-Compass
Student Compass is a web application built with Streamlit that helps students discover educational opportunities from around the world.
The application provides a searchable database of:
  Online courses
  Competitions
  Summer programs
  Research opportunities
  Volunteering
  Tutoring
  Extracurricular activities
  Features
🔍 Search by title, description, or tags
🎯 Filter by difficulty level
💰 Filter by cost
📂 Filter by category
🌍 Direct links to official resources
Technologies:
  Python
  Streamlit
  Pandas
  Google Sheets (CSV)
Why? 
  I created this project to make it easier for students to find high-quality educational opportunities in one place. Instead of searching across multiple websites, users can browse and filter resources through a simple interface.

Future Improvements:
 Favorites system
 Resource recommendations
 Deadline tracking
 User accounts
 Better search and ranking
How to Run?
  pip install -r requirements.txt
  streamlit run app.py
